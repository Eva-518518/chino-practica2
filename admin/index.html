Cómo subir a GitHub Pages
1
Ve a tu repositorio en github.com y abre la carpeta data/.
2
Haz clic en lessons.json → botón ✏️ editar → pega el contenido exportado → "Commit changes".
3
GitHub Pages actualiza el sitio automáticamente en ~30 segundos. Los estudiantes verán las nuevas lecciones al recargar la página.
💡 Consejo: también puedes arrastrar el archivo lessons.json directamente a la carpeta data/ en GitHub usando la opción "Add file → Upload files".
Estructura de archivos en GitHub
Tu repositorio debe tener esta estructura:
tu-repo/ ├── index.html ← sitio del estudiante ├── admin/ │ └── index.html ← este panel └── data/ └── lessons.json ← aquí van todas las lecciones
En GitHub, ve a Settings → Pages → Source: main branch / root para activar GitHub Pages. Tu sitio estará en https://tu-usuario.github.io/tu-repo.
